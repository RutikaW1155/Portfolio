Amazon Clone

An Amazon Clone is a beginner project where you replicate Amazon's layout using HTML and CSS. It includes a homepage with product listings, a navigation bar, and a simple product page. This project helps you practice basic web development skills like structure and styling.

Heart disease health indicator

The *Heart Disease Health Indicators* project uses exploratory data analysis (EDA) to explore health data and identify key factors like cholesterol, blood pressure, and lifestyle habits that influence the risk of heart disease.

Simon Game

The Simon Game is a classic memory game where players must repeat a sequence of colors and sounds. It starts with the game lighting up a series of buttons, and players replicate the sequence in the correct order.

The Drum Kit is an interactive music application that allows users to create and play beats using various drum sounds. Featuring a set of virtual drum pads for kick drums, snare drums, hi-hats, and cymbals, it provides a user-friendly interface and high-quality sound variety.
